# 7주차 실습 기록

## 오늘 한 것
기존에 제작한 Breakout 게임 코드를 기반으로,
시각적 연출과 에셋 최적화를 중심으로 기능을 확장하였다.

기존의 단순 도형(draw.rect) 기반 그래픽을 이미지 에셋으로 교체하고,
사운드 및 애니메이션 요소를 추가하여 게임의 완성도를 높이는 것을 목표로 진행하였다.

## 사용한 AI 프롬프트 (요약)

1. 이미지 에셋 적용 방식 질문
   블록을 draw.rect 대신 이미지로 교체하면서,
   성능 저하 없이 효율적으로 렌더링하는 방법을 질문하였다.

2. 사운드 시스템 분리
   효과음과 배경음악을 구분하여 상황에 따라 다르게 재생하는 방법을 질문하였다.

3. 게임 밸런스 조정
   콤보 시스템과 아이템 드랍 확률을 더 쉽게 조정하여
   플레이 경험을 개선하는 방법을 질문하였다.

## AI 답변에서 도움이 된 것

- 이미지 에셋을 미리 로드하고 blit으로 출력하는 구조를 이해하였다.
- 효과음(Sound)과 배경음악(music)의 차이를 구분할 수 있었다.
- 콤보, 피버 타임 등의 조건을 변수로 조정하는 방법을 익혔다.
- 기존 코드 구조를 유지하면서 기능을 확장하는 방향을 잡을 수 있었다.

## AI 답변을 수정하거나 버린 것

AI가 제안한 일부 기능은 코드 구조와 맞지 않아 그대로 사용하지 않고 수정하였다.
특히 과도한 연출이나 복잡한 기능은 제거하고, 핵심 기능 위주로 단순화하였다.
또한 여러 AI의 답변을 참고하여 필요한 부분만 선택적으로 적용하였다.
