# 5주차 실습 기록

## 선택한 템플릿 및 주제
수업에서 제공된 Breakout 기본 코드를 템플릿으로 사용하였다.  
기존 코드에는 패들, 공, 벽돌, 레벨 시스템 등이 이미 구현되어 있었기 때문에, 이를 기반으로 게임의 구조를 이해하고 차별화된 기능을 추가하는 방향으로 진행하였다.



## 사용한 AI 프롬프트 (요약)

1. 코드 구조 설명 요청  
제공된 Breakout 코드에서 main 루프, 충돌 처리, 레벨 전환 등이 어떻게 동작하는지 이해하기 위해 질문하였다.

2. 기획서 구체화 요청  
기본 Breakout 구조를 유지하면서 어떤 요소를 추가하면 게임을 더 흥미롭게 만들 수 있는지 질문하였다.

3. 구현 가능성 확인  
아이템, 피버 시스템, 벽돌 추가 등의 기능이 현재 코드 구조에서 구현 가능한지 확인하였다.



## AI 답변에서 도움이 된 것

- main 함수 내부에서 게임이 반복되는 구조를 이해하는 데 도움이 되었다.  
- 공과 패들, 벽돌 간의 충돌 처리 흐름을 파악할 수 있었다.  
- 레벨 시스템과 블록 생성 방식이 어떻게 구성되어 있는지 이해하였다.  
- 기존 코드를 유지하면서 기능을 추가하는 방식으로 접근할 수 있었다.  



## AI 답변을 수정하거나 버린 것

 AI가 제안한 일부 기능은 현재 코드 구조와 맞지 않아 그대로 사용하지 않고 수정하였다.  
 처음에는 많은 기능을 한 번에 추가하려 했지만, 기존 코드 이해를 우선으로 하고 점진적으로 수정하는 방식으로 변경하였다.  
 일부 복잡한 기능은 구현 난이도를 고려하여 제외하거나 간단한 형태로 변경하였다.
